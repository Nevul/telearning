Contenido de tablas

¿Qué es el CSS?

Como hemos mencionado antes, CSS es un lenguaje para especificar cómo se presentan los documentos a los usuarios: cómo se diseñan, distribuyen, etc.

Un documento suele ser un archivo de texto estructurado con un lenguaje de marcado : HTML es el lenguaje de marcado más común, pero también encontrará otros lenguajes de marcado como SVG o XML .

Presentar un documento a un usuario significa convertirlo en una forma utilizable para su audiencia. Los navegadores , como Firefox , Chrome o Internet Explorer , están diseñados para presentar documentos visualmente, por ejemplo, en una pantalla de computadora, proyector o impresora.

¿Cómo afecta CSS a HTML?

Los navegadores web aplican reglas CSS a un documento para afectar cómo se muestran. Una regla CSS se forma a partir de:

  • Un conjunto de propiedades , que tienen valores establecidos para actualizar la forma en que se muestra el contenido HTML, por ejemplo, quiero que el ancho de mi elemento sea el 50% de su elemento principal y que su fondo sea rojo.
  • Un selector , que selecciona los elementos a los que desea aplicar los valores de propiedad actualizados. Por ejemplo, quiero aplicar mi regla CSS a todos los párrafos de mi documento HTML.

Un conjunto de reglas CSS contenidas en una hoja de estilo determina cómo debe verse una página web.

Un toque de vocabulario

En su nivel más básico, CSS consta de dos bloques de construcción:

  • Propiedades: identificadores legibles por humanos que indican qué características estilísticas (por ejemplo, fuente, ancho, color de fondo) desea cambiar.
  • Valores: a cada propiedad especificada se le asigna un valor que indica cómo desea cambiar esas características estilísticas (por ejemplo, a qué desea cambiar la fuente, el ancho o el color de fondo).

Una propiedad emparejada con un valor se denomina declaración CSS . Las declaraciones CSS se colocan dentro de los bloques de declaración CSS . Y, por último, los bloques de declaración CSS se combinan con selectores para producir conjuntos de reglas CSS (o reglas CSS ).

Sintaxis en detalle

Declaraciones CSS

Establecer las propiedades de CSS en valores específicos es la función principal del lenguaje CSS. El motor CSS calcula qué declaraciones se aplican a cada elemento de una página para diseñarla y diseñarla adecuadamente. La propiedad y el valor en cada par están separados por dos puntos (:).

Hay más de 300 propiedades diferentes en CSS y casi un número infinito de valores diferentes. No todos los pares de propiedades y valores están permitidos. Cada propiedad tiene una lista específica de valores válidos definidos para ella.

Bloques de declaración CSS

Las declaraciones se agrupan en bloques, con cada conjunto de declaraciones envuelto por una llave de apertura ( { ) y una de cierre ( } ).

Las declaraciones dentro de un bloque de declaración deben estar separadas por punto y coma ( ; ), de lo contrario, el código no funcionará (o al menos dará resultados inesperados). La última declaración de un bloque no necesita ser terminada por un punto y coma. Sin embargo, a menudo se considera un buen estilo hacerlo, ya que evita olvidar agregarlo al extender el bloque con otra declaración.

Selectores CSS y reglas

Nos falta una parte del rompecabezas: necesitamos analizar cómo decirle a nuestros bloques de declaración a qué elementos se deben aplicar. Esto se hace prefijando cada bloque de declaración con un selector , un patrón que coincide con algunos elementos de la página. Las declaraciones asociadas se aplicarán solo a esos elementos. El selector más el bloque de declaración se denomina conjunto de reglas o, a menudo, simplemente una regla.

Los selectores pueden volverse muy complicados: puede hacer que una regla coincida con varios elementos al incluir múltiples selectores separados por comas (un grupo), y los selectores se pueden encadenar juntos.

Un elemento puede coincidir con varios selectores, por lo tanto, varias reglas pueden establecer una propiedad determinada varias veces. CSS define cuál tiene prioridad sobre las demás y debe aplicarse.

Introducción de selectores

Los selectores se usan para apuntar a los elementos HTML en nuestras páginas web que queremos diseñar. Hay una amplia variedad de selectores CSS disponibles, lo que permite una precisión de grano fino al seleccionar elementos para diseñar.

Los básicos

En la última sección explicamos en detalle la sintaxis y la terminología general de CSS. Para recapitular, los selectores son una parte de una regla CSS y vienen justo antes de que se bloqueen las declaraciones CSS.

Diferentes tipos de selectores

Los selectores se pueden dividir en las siguientes categorías:

  • Selectores simples: haga coincidir uno o más elementos según el tipo de elemento, clas o id.
  • Selectores de atributos: unen uno o más elementos en función de sus atributos / valores de atributo.
  • Pseudo-clases: coincide con uno o más elementos que existen en un determinado estado, como un elemento que se encuentra sobre el puntero del mouse, o una casilla de verificación que está actualmente desactivada o marcada, o un elemento que es el primer elemento secundario de su padre en el árbol DOM.
  • Pseudoelementos: coincide con una o más partes de contenido que están en una posición determinada en relación con un elemento, por ejemplo, la primera palabra de cada párrafo, o contenido generado que aparece justo antes de un elemento.
  • Combinadores: estos no son exactamente selectores en sí mismos, sino formas de combinar dos o más selectores de manera útil para selecciones muy específicas. Entonces, por ejemplo, puede seleccionar solo párrafos que sean descendientes directos de divs, o párrafos que vienen directamente después de los encabezados.
  • Selectores múltiples: una vez más, estos no son selectores separados; La idea es que puede colocar múltiples selectores en la misma regla CSS, separados por comas, para aplicar un único conjunto de declaraciones a todos los elementos seleccionados por esos selectores.
