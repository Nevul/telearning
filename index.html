Qué es Html?

HyperText Markup Language es el componente básico de la Web. Define el significado y la estructura del contenido web.

"Hipertexto" se refiere a enlaces que conectan páginas web entre sí, ya sea dentro de un único sitio web o entre sitios web. Los enlaces son un aspecto fundamental de la Web.

Qué es Css?

Cascading Style Sheets (CSS) es un lenguaje de hojas de estilo utilizado para describir la presentación de un documento escrito en HTML o XML.

CSS describe cómo se deben representar los elementos en la pantalla, en papel, en el habla o en otros medios.
