Introducción a Html

Conceptos básicos de Html, para comenzar. Definimos elementos y atributos. También mostramos como se estructura un elemento Html y demás términos importantes.

Anatomía de un elemento html

Las partes principales de nuestro elemento son:

  1. La etiqueta de apertura: Consiste en el nombre del elemento (en este caso, p), envuelto en paréntesis angulares de apertura y cierre. Esto indica dónde comienza a surtir efecto el elemento, en este caso dónde está el comienzo del párrafo.
  2. La etiqueta de cierre: es la misma que la etiqueta de apertura, excepto que incluye una barra diagonal antes del nombre del elemento. Esto indica dónde termina el elemento, en este caso dónde está el final del párrafo. No incluir una etiqueta de cierre es un error común de principiante y puede dar lugar a resultados extraños.
  3. El contenido: este es el contenido del elemento, que en este caso es solo texto.
  4. El elemento: la etiqueta de apertura más la etiqueta de cierre más el contenido es igual al elemento.

Bloque versus elementos en línea

Hay dos categorías importantes de elementos en HTML que debe conocer. Son elementos a nivel de bloque y elementos en línea.

  • Los elementos de nivel de bloque forman un bloque visible en una página: aparecerán en una nueva línea desde cualquier contenido que haya antes, y cualquier contenido que vaya después también aparecerá en una nueva línea. Los elementos a nivel de bloque tienden a ser elementos estructurales en la página que representan, por ejemplo, párrafos, listas, menús de navegación, pies de página, etc. Un elemento a nivel de bloque no estaría anidado dentro de un elemento en línea, pero podría estar anidado dentro otro elemento a nivel de bloque.
  • Los elementos en línea son aquellos que están contenidos dentro de elementos de nivel de bloque y rodean solo pequeñas partes del contenido del documento, no párrafos enteros y agrupaciones de contenido. Un elemento en línea no hará que aparezca una nueva línea en el documento; normalmente aparecerían dentro de un párrafo de texto, por ejemplo, un elemento <a> (hipervínculo) o elementos de énfasis como <em> o <strong>.

Atributos

Los elementos también pueden tener atributos, que se ven así:

Los atributos contienen información adicional sobre el elemento que no desea que aparezca en el contenido real. En este caso, el atributo de "class" permite dar al elemento un nombre de identificación que luego puede usarse para apuntar al elemento con información de estilo y otras cosas.

Un atributo debe tener:

  1. Un espacio entre este y el nombre del elemento (o el atributo anterior, si el elemento ya tiene uno o más atributos).
  2. El nombre del atributo, seguido de un signo igual.
  3. Un valor de atributo, con comillas de apertura y cierre envuelto alrededor de él.

Agregar atributos a un elemento

Otro ejemplo de un elemento es <a> : significa "ancla" y convertirá el fragmento de texto en un hipervínculo. Esto puede tomar una serie de atributos, pero varios son los siguientes:

  • href: este atributo especifica como su valor la dirección web a la que desea que apunte el enlace; donde navega el navegador cuando se hace clic en el enlace. Por ejemplo, href="https://www.mozilla.org/".
  • title: el atributo de title especifica información adicional sobre el enlace, como por ejemplo a qué página está enlazando. Por ejemplo, title="The Mozilla homepage" . Esto aparecerá como información sobre herramientas al pasar el cursor sobre él.
  • target: atributo target especifica el contexto de navegación que se utilizará para mostrar el enlace. Por ejemplo, target="_blank" mostrará el enlace en una nueva pestaña. Si desea mostrar el enlace en la pestaña actual, simplemente omita este atributo.

Atributos booleanos

A veces verá atributos escritos sin valores, esto está perfectamente permitido. Estos se llaman atributos booleanos y solo pueden tener un valor, que generalmente es el mismo que el nombre del atributo. Como ejemplo, tome el atributo disabled, que puede asignar a los elementos de entrada del formulario si desea que estén deshabilitados (atenuados) para que el usuario no pueda ingresar ningún dato en ellos.

Anatomía de un documento HTML.

Eso resume los conceptos básicos de elementos HTML individuales, pero no son muy útiles por sí solos. Ahora veremos cómo se combinan los elementos individuales para formar una página HTML completa:

Aquí tenemos:

  1. <!DOCTYPE html> es la cadena de caracteres más corta que cuenta como un tipo de documento válido.
  2. <html></html> Este elemento envuelve todo el contenido de toda la página, y a veces se lo conoce como el elemento raíz.
  3. <head></head> Este elemento actúa como un contenedor para todas las cosas que desea incluir en la página HTML que no es el contenido que está mostrando a los visitantes de su página.
  4. <meta charset="utf-8">: Este elemento establece el conjunto de caracteres que su documento debe usar para UTF-8, que incluye la mayoría de los caracteres de la gran mayoría de los idiomas escritos humanos.
  5. <title></title> Esto establece el título de su página, que es el título que aparece en la pestaña del navegador en el que se carga la página, y se utiliza para describir la página cuando la marca como favorita.
  6. <body></body> Contiene todo el contenido que desea mostrar a los usuarios web cuando visitan su página, ya sea texto, imágenes, videos, juegos, pistas de audio reproducibles o cualquier otra cosa.
